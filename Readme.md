
# Node TextMate Bundle

 Simple [Node](http://nodejs.org) TextMate bundle, feel
 free to fork!

## Snippets

Below are _some_ of the tab completion snippets provided:

    `try`   - try catch block
	`st`    - setTimeout()
	`si`    - setInterval()
	`ct`    - clearTimeout()
	`ci`    - clearInterval()
	`r`     - require()
	`se`    - setEncoding()
	`i`     - sys.inherit()
	`um`    - process.umask()
	`suid`  - process.setuid()
	`sgid`  - process.setgid()
	`kill`  - process.kill()
	`cwd`   - process.cwd()
	`cd`    - process.chdir()
	`exit`  - process.exit()
	`al`    - addListener()
	`ok`    - Object.keys()
	`p`     - require('sys').puts(require('sys').inspect(obj))
	`rf`    - fs.readFile()
	`rfs`    - fs.readFileSync()
